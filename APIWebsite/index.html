Welcome to the Summarize PDF API Documentation

This API is 'open source' and makes use of OpenAI (Documentation found here), which means that you are free to consume it for anything you want.

The API has 4 endpoints, Sign Up, Login, Summarize and Credits all of which can be consumed

This API uses the FastAPI library, (documentation found here).

Sign Up:

This endpoint requires the name, email address and password parameters

User information is then entered into the MongoDB

Signin:

This endpoint requires the email address and password parameters

User information is then checked against the database to authenticate the user

Summarize

This endpoint is protected by JWTBearer tokens that are sent with authenticated users

Once authenticated, PDF files can be uploaded and sent to ChatGPT for summarization

Credits

This endpoint defines the amount of times a user can carry out a summarization request

Each new user gets given 3 credits to use
